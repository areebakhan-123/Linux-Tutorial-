Linux Basic Tutorial

1. Navigating Directories

Command: cd

Description: Use cd to move between folders/directories.

2. Listing Files

Command: ls

Description: Lists files and folders in the current directory.

3. Creating a File

Command: touch filename.txt

Description: Creates a new empty file.

4. Creating a Directory

Command: mkdir foldername

Description: Makes a new folder.

5. Deleting Files and Directories

Commands: rm filename.txt (file), rm -r foldername (directory)

Description: Removes files or folders.

6. Viewing File Contents

Commands: cat filename.txt or less filename.txt

Description: Displays the content of a file.

7. Copying and Moving Files

Commands: cp source destination, mv source destination

Description: cp copies files. mv moves or renames them.

8. Checking Current Path

Command: pwd

Description: Shows the full path of the current working directory.

9. Installing Software

Command: sudo apt install packagename (Debian/Ubuntu)

Description: Installs new software packages.

10. Checking Running Processes

Command: top or ps

Description: Monitors active processes and resource usage.
